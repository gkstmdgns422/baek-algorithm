## 4️⃣ Gitlab 코드 관리

### 4. 코드 작성 후 Github 제출

1. git bash의 `algorithm/` 위치에서 풀이한 문제를 `git add <풀이한문제>` 로 git staging area에 추가

   - git status 로 git stage 상태 확인

     

2. `git commit -m <commit_message>` 로 커밋 기록 생성

   

3. `git push` 으로 github에 push
